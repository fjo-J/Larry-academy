Mindset

今の自分を「拡張」するスキル

AIエンジニアに転職する必要はありません。今のあなたの専門知識にAIという武器を掛け合わせることで、インプットの質を飛躍させ、可能性を劇的に広げるのがLarryの目的です。

AIは「空気が読めない新入社員」

1回のプロンプトで完璧な答えを出させようとするのはやめましょう。まずは60点の叩き台を作らせ、ツッコミ（ラリー）を入れて育て上げるのがプロの思考です。

Preparation

AIのアイデアを映し出す「鏡」を用意する

プログラミングの勉強は不要ですが、AIが出力したコードをブラウザで表示させるための「プレビュー環境」は必須です。世界中のプロが使う無料の作業台「VS Code」を導入します。

1. VS Code と Live Server の導入

公式サイトからVS Codeをインストールし、拡張機能アイコンから「Live Server」を検索して追加します。

公式サイトを開く

2. 【重要】必ず「フォルダ」ごと開く

空のフォルダを作り、VS Codeの「フォルダーを開く」から読み込みます。ファイルだけを直接作るとプレビュー機能が動きません！

Anti-Patterns

初心者が必ず引っかかる「3つのAIの罠」

罠①：「見た目」だけを整えさせる

プロのラリー：実際に動かすための「裏側の仕組み（ノウハウDB）」を組み込んで作り直して。

罠②：「空気を読んで」機能を足させる

プロのラリー：AIに空気を読ませず、具体的な利用シーンを添えてピンポイントで指示する。

罠③：すべてを「一発」で終わらせようとする

プロのラリー：タスクを分割し、まずは骨組みだけを出力させ、一歩ずつ深掘りする。

意図を的確に伝える「コンテキスト」

AIはあなたの会社の事情を知りません。「誰が」「どんな状況で」「何のために」使うのかという背景情報を与えるだけで、出力の解像度が劇的に上がります。

Context_Template.txt
以下の要件に従って、[ 作成したいもの ] を作成してください。
【コンテキスト】
・私の役割：[ マネージャー ]
・ターゲット：[ 現場スタッフ ]
・目的：[ 報告の提出率を上げること ]

出力形式のコントロール術

AIにアウトプットの『器』を強制的に指定するテクニックです。「Markdown形式で」「表形式で比較して」など、後で見直しやすい形で出させます。

例：比較検討させたい時

3つの案を提案し、それぞれのメリット・デメリットを【表形式】で出力してください。

構成案を壁打ちで作る

いきなりコードを書かせるのではなく、まずは「ターゲットの悩み」「解決策」といったLPの骨組みをAIと一緒に作ります。

Prompt_LP1.txt
あなたはプロのWebマーケターです。
[ 商品名 ] のLPを作成します。まずは以下の要素を含んだ構成案を作成してください。
1. ターゲットの悩み、2. キャッチコピー、3. 行動喚起(CTA)

コピーのトーンを調教する

Prompt_LP2.txt
構成案は素晴らしいです。ただ、文章が少しお堅いです。
「小学生でもわかるように」「ワクワクさせる情熱的な表現」で書き直してください。

箇条書きから完璧な企画書へ

【構成】1. 背景、2. 提案、3. 効果

既存データのエラー修正

Prompt_Fix.txt

どこが悪いか指摘した上で、修正した完璧なコードを出力してください。

インプットを「評価」に変える

質の高い成果物を「URLやPDF」にして外へ届けるスキルを学びます。今のあなたの価値を爆増させる最後のピースです。

URL化の魔法 (GitHub)

  • 1. GitHubでリポジトリを作る。
  • 2. `index.html` をドロップして保存。
  • 3. Pagesを有効にする。これでURLが発行されます。

完璧な資料化 (PDF)

Prompt_PDF.txt
印刷時に背景色が保持され、A4サイズで改ページが綺麗に行われるような「@media print」のCSSを追加してください。
